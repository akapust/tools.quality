Name Tool

Ishikawa-Diagramm (Fischgräte) – 6M

Ziel

Ursachen strukturiert sammeln, clustern und visualisieren, um den wichtigsten Einflussfaktoren auf den Effekt näherzukommen.

So benutzt du das Tool

  1. Effekt / Problem rechts im Kopf eintragen.
  2. Pro 6M-Kategorie Ursachen hinzufügen.
  3. Bei Bedarf Ursachen einzeln löschen (X).
  4. Export als PNG oder als JSON speichern.

Datenschutz

Dieses Tool arbeitet vollständig im Browser. Es werden keine Daten auf einem Server gespeichert. Zugriff hat nur die Person, die die Datei lokal speichert bzw. lädt.
